# Artificial Intelligence & Machine Learning

This repository contains comprehensive AI and machine learning projects demonstrating proficiency in various AI techniques, from classical machine learning to deep learning and reinforcement learning.

## Projects Overview

### Machine Learning & Statistics
- **01-Naive-Bayes-Classification-Language-Detection** - Probabilistic classification for language detection
- **02-Principal-Component-Analysis-PCA-Dimensionality-Reduction** - Dimensionality reduction and feature extraction
- **03-Hierarchical-Agglomerative-Clustering-HAC** - Unsupervised learning and clustering algorithms
- **04-Linear-Regression-Statistical-Modeling** - Statistical modeling and regression analysis

### Deep Learning & Neural Networks
- **05-Introduction-to-PyTorch-Neural-Networks** - Deep learning fundamentals with PyTorch
- **06-Convolutional-Neural-Networks-LeNet-Architecture** - CNN implementation and computer vision

### Search Algorithms & Game AI
- **07-A-Star-Search-Algorithm-8-Puzzle-Solver** - A* search algorithm for puzzle solving
- **08-Minimax-Algorithm-Game-AI-Teeko-Player** - Game AI using minimax algorithm
- **09-Q-Learning-Reinforcement-Learning-FrozenLake** - Reinforcement learning with Q-learning
